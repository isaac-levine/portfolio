# isaac-levine.com

My personal portfolio and blog, built with Next.js and MDX.

## Tech Stack

- **Framework:** Next.js 13, React 18, TypeScript
- **Content:** MDX with remark-gfm
- **Styling:** Tailwind CSS, Framer Motion
- **Code:** Prism.js syntax highlighting, Mermaid diagrams
- **Analytics:** Vercel Analytics
- **Deployment:** Vercel

## Features

- Project showcase
- Blog with MDX support and code highlighting
- Resume page
- Contact form
- Full-text search with pre-built index

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
