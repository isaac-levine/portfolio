- **View build logs:** `get_deployment_build_logs` to investigate build failures
- **View runtime logs:** `get_runtime_logs` to debug application errors in production
- **Fetch deployed page:** `web_fetch_vercel_url` to access the deployed site (handles Vercel auth)
